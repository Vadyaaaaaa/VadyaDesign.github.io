=== ИНСТРУКЦИЯ ПО УСТАНОВКЕ ===

СПОСОБ 1 - Автоматическая установка:
1. Запустите install_extensions.bat (Windows) или install_extensions.sh (Linux/Mac)
2. Дождитесь завершения установки
3. Перезапустите Cursor

СПОСОБ 2 - Ручная установка через Command Palette:
1. Нажмите Ctrl+Shift+P
2. Введите "Extensions: Install Extensions"
3. Найдите каждое расширение по ID (например: ms-python.python)
4. Нажмите Install

СПОСОБ 3 - Через боковую панель:
1. Нажмите Ctrl+Shift+X
2. Введите название расширения в поиск
3. Нажмите Install

СПОСОБ 4 - Через терминал (если доступен CLI):
cursor --install-extension ms-python.python
cursor --install-extension ms-python.vscode-pylance
# и так далее для каждого расширения
